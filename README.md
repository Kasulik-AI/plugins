# Kasulik AI Plugins

37 AI skills + 20 commands packaged into 5 plugins for small and medium businesses. Install them in [Claude](https://claude.ai) via Cowork or Claude Code.

**Uus kasutaja?** Loe kõigepealt [KIIRSTART-JUHEND.md](KIIRSTART-JUHEND.md) ja [TURVALISUS-JUHEND.md](TURVALISUS-JUHEND.md).

## Quick Start

```
/plugin marketplace add kasulik-ai/plugins
/plugin install kasulik-ai-workflows@kasulik-ai-plugins
/plugin install kasulik-ai-marketing-sales@kasulik-ai-plugins
/plugin install kasulik-ai-operations-support@kasulik-ai-plugins
/plugin install kasulik-ai-strategy-leadership@kasulik-ai-plugins
/plugin install kasulik-ai-turism@kasulik-ai-plugins
```

Then run `/alusta` to set up your workspace.

## How It Works

1. **Install all 5 plugins** (they work best together)
2. **Run `/alusta`** — guides you through setup
3. **Run `/agendid`** to create a personalized AI assistant
4. **Run `/kontekst`** to set your marketing context
5. **Just describe what you need** — skills trigger automatically
6. **Schedule `/review` weekly** to improve your workspace
