# kasulik-turundus

8 turundusoskust väike- ja keskmistele ettevõtetele: turunduskontekst, copywriting (25+ psühholoogiamudeliga), e-posti kampaaniad, SEO, tasulised reklaamid, kampaaniaplaneerimine, hinnastrateegia ja käivitusstrateegiad.

## Quick Start

Käivita esmalt `/kontekst` — see seadistab su ettevõtte turunduskonteksti (positsioneerimine, sihtrühm, brändihääl), mida teised skillid omakorda kasutavad.

## Käsud

| Käsk | Kirjeldus |
|------|-----------|
| `/kontekst` | Turunduskonteksti seadistamine — sihtrühm, positsioneerimine, konkurendid, brändihääl, väärtuspakkumine |
| `/tekst` | Turundus- ja müügitekstid 25+ psühholoogiamudeliga — koduleht, sotsiaalmeedia, reklaamid, blogipostitused |
| `/meilid` | Meilikampaaniad ja -mallid — tervitusmeilid, müügijärjestused, uudiskirjad, järeltegevused |
| `/seo` | SEO audit ja optimeerimine — tehniline SEO, märksõnauuringud, sisu optimeerimine, lokaalne SEO |
| `/reklaam` | Tasuliste reklaamide strateegia — Google Ads, Meta, LinkedIn, TikTok |
| `/kampaania` | Turunduskampaaniate planeerimine — eesmärgid, sihtrühm, sõnumid, kanalite strateegia, sisukalender |
| `/hind` | Hinnastrateegia ja pakettide kujundamine — väärtuspõhine hinnastamine, tasemete struktuur, freemium |
| `/käivitus` | Toote ja teenuse turuletoomine — ORL raamistik (Oma/Renditud/Laenatud kanalid), faasid |

## Skillid

### kontekst
Turunduskonteksti vundamendi loomine. Kõik teised turundusoskused ehitavad selle peale.

### tekst
Copywriting 25+ psühholoogiamudeliga (ankurdamine, kaotusekartus, sotsiaalne tõendus, raamistamine, vastastikkus jne). Kõik suuremad tekstitüübid: koduleht, landing page, reklaamid, blogi, tooted.

### meilid
Meilikampaaniate disain — tervitusjada, müügijärjestus, uudiskiri, A/B testimine, automatiseeritud vood.

### seo
SEO audit + optimeerimine: tehniline SEO, märksõnauuringud, sisuoptimeerimine, lokaalne SEO, schema markup.

### reklaam
Tasuliste reklaamide strateegia ja optimeerimine 4 platvormil (Google, Meta, LinkedIn, TikTok). Eelarveteadlik, mitmekeelne sihtimine (eesti/inglise/vene).

### kampaania
Terviklikud turunduskampaaniad eesmärkidest kuni sisukalendri ja KPI-deni.

### hind
Hinnastrateegia: väärtuspõhine hinnastamine, tasemete struktuur, freemium vs prooviaeg, hinnatõusu kava.

### käivitus
Toote/teenuse turuletoomise strateegia ORL raamistikuga (Oma, Renditud, Laenatud kanalid), faasipõhine lähenemine, hoogu hoidmine.

## Töötab kõige paremini koos

- **kasulik-tööruum** — `/kontekst` salvestab info memory'sse, mida teised pluginad kasutavad
- **kasulik-müük** — pärast turunduskonteksti `/pöördumine` ja `/prospekt` töötavad oluliselt paremini
- **kasulik-strateegia** — `/plaan` ja `/kasv` raamistavad turundusotsuseid
