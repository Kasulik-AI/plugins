# kasulik-ai-workflows

AI workspace setup, coaching, AI assistant creation, and continuous improvement for SMBs. The foundation plugin for the Kasulik AI SMB Skills Collection.

## Quick Start

Run `/alusta` — it detects your current state and adapts:
- **New workspace?** → Guided setup in 5-10 minutes
- **Basic setup done?** → Offers to deepen and customize
- **Already configured?** → Coaching mode with skill recommendations

## Commands

| Command | Description |
|---------|-------------|
| `/alusta` | Smart entry point — setup, improvement, or coaching based on workspace state |
| `/agendid` | Create a personalized AI assistant — interview, auto-scan, layered output |
| `/workflows` | Browse multi-skill workflow recipes for common business tasks |
| `/töövood` | Töövoo retseptid — 12 eelseadistatud mitmeskillset töövoogu |
| `/review` | Weekly 15-minute reflection to improve your AI usage |
| `/ülevaade` | Eestikeelne iganädalane ülevaade |
| `/tabel` | Exceli ja CSV failide analüüs |
| `/wordpress` | WordPress MCP integreerimise juhend |

## Skills

### workspace-setup
Adaptive onboarding: interviews you about your business, researches your website, generates CLAUDE.md and memory files, optionally creates a custom skill for your workflows.

### ai-coach
Ongoing coaching: finds the right skill for any task, teaches workflow recipes, improves your prompting, and builds strategic AI confidence.

### weekly-review
Structured weekly check-in: celebrates wins, diagnoses struggles, suggests new skills to try, and keeps your workspace context up to date.

### agendid
AI assistant creation and training: 6-phase workflow (interview → auto-scan → synthesis → generation → presentation) that creates a layered AI workspace — Project Instructions + CLAUDE.md + memory/ folder + ChatGPT GPT export + Claude Projects export.

## Works Best With

This plugin is designed to work alongside the other Kasulik AI plugins:
- **kasulik-ai-marketing-sales** — 20 marketing, sales, and brand skills
- **kasulik-ai-operations-support** — 9 operations and customer support skills
- **kasulik-ai-strategy-leadership** — 10 strategy, product, and leadership skills
- **kasulik-ai-turism** — 5 tourism-specific skills

The ai-coach skill knows about all skills across these plugins and will recommend the right one for any task.
