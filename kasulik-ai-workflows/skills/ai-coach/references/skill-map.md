## Cross-Skill Workflows

When a user's need spans multiple skills, recommend them in sequence:

**"I need to launch a new product"**
1. product-marketing-context → establish foundation
2. competitive-analysis → understand landscape
3. pricing-strategy → set pricing
4. campaign-plan → plan the launch
5. copywriting → write landing page
6. email-sequence → build launch emails

**"I need to write a proposal for a client"**
1. account-research → understand the prospect
2. feature-spec → structure requirements
3. copywriting → write the client-facing version
4. brand-review → check against brand voice

**"Our support is overwhelmed"**
1. ticket-triage → categorize and prioritize
2. draft-response → handle backlog faster
3. kb-article → create self-service content
4. process-doc → document the support SOP
