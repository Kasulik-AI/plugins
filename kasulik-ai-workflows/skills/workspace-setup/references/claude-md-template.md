# CLAUDE.md Template

Use this template when generating CLAUDE.md for a new workspace. Replace all `{placeholders}` with actual values from the interview and research.

```markdown
# {Business Name} — AI Workspace

## About
{One paragraph: what the business does, who they serve, what makes them different.}

## Team Context
- **User**: {Name}, {Role}
- **Primary AI use cases**: {list of 2-4 use cases}
- **Tone**: {derived from website — e.g., "Professional but approachable, practical over theoretical"}

## How to Help Me

When I ask for help with:
- **Marketing/content** → Use copywriting, draft-content, email-sequence, or seo-audit skills
- **Sales/outreach** → Use draft-outreach, call-prep, or account-research skills  
- **Operations** → Use process-doc, runbook, or status-report skills
- **Strategy** → Use feature-spec, competitive-analysis, or roadmap-management skills
- **Writing** → Always apply writing-clearly-and-concisely principles

## Business Details
- **Website**: {url}
- **Industry**: {industry}
- **Target audience**: {who they sell to}
- **Key products/services**: {list}

## Preferences
- {Any preferences discovered during interview — language, format, level of detail}
- Always ask clarifying questions before starting substantial work
- Use TodoWrite for multi-step tasks

## Installed Skills
{Auto-detected list of Kasulik plugins and their key skills}
```

## Adaptation Notes

- If user has no website, skip the website section and ask more interview questions
- If user selected specific Kasulik plugins, only list skills from those plugins in "How to Help Me"
- If user mentioned specific tone/voice preferences, override the website-derived tone
- For Estonian businesses: note if they prefer Estonian or English output
