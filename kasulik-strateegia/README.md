# kasulik-strateegia

5 strateegia- ja juhtimisoskust väike- ja keskmistele ettevõtetele: strateegilised plaanid, aruanded ja ülevaated, protsesside dokumenteerimine, ärikasvu strateegiad ja konkurentsianalüüs.

## Quick Start

Sõltub eesmärgist:
- **Kvartaliplaan või OKR-id** → `/plaan`
- **Juhatuse uuendus, kuuaruanne** → `/aruanne`
- **SOP, töövoo dokumenteerimine** → `/protsess`
- **Skaleerimine, uued turud** → `/kasv`
- **Konkurentsi-uuring, eristumine** → `/konkurent`

## Käsud

| Käsk | Kirjeldus |
|------|-----------|
| `/plaan` | Strateegilised plaanid — aasta eesmärgid, SWOT, tegevuskava, hooajaplaan, OKR-id, ressursiplaan |
| `/aruanne` | Aruanded ja ülevaated — olekuraport, finantsaruanne, juhatuse uuendus, kvartaalne ülevaade, stakeholder-kommunikatsioon |
| `/protsess` | Äriprotsesside dokumenteerimine ja optimeerimine — SOP, RACI maatriks, kitsaskohad, automatiseerimine |
| `/kasv` | Ärikasvu strateegiad — skaleerimine, uued turud, partnerlused, innovatsioon, tuluvood, laienemine |
| `/konkurent` | Konkurentsianalüüs ja positsioneerimine — konkurentide kaardistamine, SWOT, eristumisstrateegiad, turupositsioon |

## Skillid

### plaan
Strateegiline planeerimine: aastaplaan, kvartaliplaan, hooajaplaan, OKR-id, SWOT-analüüs, tegevuskavad. Toetub `/kontekst`-ile ja `/konkurent`-ile sisendiks.

### aruanne
Aruandlus mitmes formaadis: olekuraport (status report), juhatuse uuendus, kuuaruanne, kvartalikokkuvõte, stakeholder-kommunikatsioon.

### protsess
SOP-de, RACI maatriksite ja töövoogude dokumenteerimine. Tuvastab kitsaskohti ja pakub automatiseerimise võimalusi.

### kasv
Skaleerimise ja laienemise strateegiad: uued turud, partnerlused, innovatsioon, alternatiivsed tuluvood, ärimudelite arendamine.

### konkurent
Konkurentsianalüüs: konkurentide kaardistamine, eristumiseesmärkide määratlemine, SWOT, positsioneerimine, hinnavõrdlus.

## Töötab kõige paremini koos

- **kasulik-tööruum** — `/agendid` loodud AI assistent teab su ettevõtte konteksti, mis annab strateegiaotsustele parema sisendi
- **kasulik-turundus** — `/kontekst` ja `/konkurent` täiendavad üksteist (turunduspositsioneerimine ↔ strateegiline eristumine)
- **kasulik-müük** — `/konkurent` annab müügimeeskonnale vastuväidete käsitlemise materjali
