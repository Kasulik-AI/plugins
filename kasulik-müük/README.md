# kasulik-müük

5 müügi- ja kliendisuhtlusoskust väike- ja keskmistele ettevõtetele: kõnekokkuvõtted ja järeltegevused, kliendisuhtlus, müügikõne ettevalmistus, personaalne pöördumine ja prospektimine.

## Quick Start

Sõltub töövoo etapist:
- **Enne kõnet** → `/ettevalmistus` (uuring, agenda, vastuväited) või `/prospekt` (ettevõtte/isiku uurimine)
- **Pärast kõnet** → `/kõnekokkuvõte` (tegevuspunktid, järelkiri, sisemine kokkuvõte)
- **Külma pöördumise jaoks** → `/pöördumine` (e-kiri, LinkedIn, jada)
- **Klientide päringutele vastamiseks** → `/klient`

## Käsud

| Käsk | Kirjeldus |
|------|-----------|
| `/kõnekokkuvõte` | Kõnemärkmed/transkriptsioon → tegevuspunktid, järelkiri, sisemine kokkuvõte |
| `/klient` | Kliendisuhtluse haldamine — päringud, kaebused, KKK artiklid, vastusemallid |
| `/ettevalmistus` | Müügikõneks valmistumine — ettevõtte uurimine, osalejate taust, agenda, vastuväited |
| `/pöördumine` | Personaalne müügipöördumine — uuringule tugineva e-kirja koostamine, LinkedIn, järelkontakt |
| `/prospekt` | Ettevõtte või isiku uurimine — veebiotsing, profiil, võtmeisikud, värsked uudised, kvalifitseerimissignaalid |

## Skillid

### kõnekokkuvõte
Töötleb kõnemärkmeid, transkriptsiooni või salvestust → 3 tüüpi väljundit: kliendile suunatud järelkiri, sisemine meeskonnakokkuvõte, tegevuspunktide loend.

### klient
Kliendipäringute, kaebuste ja korduvate küsimuste käsitlemine. Vastusemallid, KKK artiklid, eskaleerimisstrateegiad.

### ettevalmistus
Müügikõneks ettevalmistamine: ettevõtte uurimine, osalejate LinkedIn-taust, soovituslik agenda, avastusküsimused, vastuväidete käsitlejad.

### pöördumine
Personaalne väljapöördumine prospektidele — uuring esmalt, siis e-kiri/LinkedIn/järelkontakti jada. Toetub `/prospekt`-i poolt kogutud informatsioonile.

### prospekt
Põhjalik müügieelne uuring ettevõtete ja isikute kohta — veebiotsing, ettevõtteprofiil, võtmeisikud, finantssignaalid, kvalifitseerimine.

## Töötab kõige paremini koos

- **kasulik-tööruum** — `/agendid` loodud AI assistent teab su ettevõtet, mis muudab kõik müügiväljundid personaalsemaks
- **kasulik-turundus** — `/kontekst` selgitab väärtuspakkumise, mille `/pöördumine` ja `/klient` kasutavad
- **kasulik-strateegia** — `/konkurent` aitab vastuväidete käsitlemisel, `/plaan` strateegilises müügis
